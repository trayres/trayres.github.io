Understanding the FFT, Zonst, 2e

Git Repo - Zonst QB64 Edition

Here we're going to talk about a book I recently discovered which I think is absolutely amazing.

The author describes FFT's in a natural way and uses GWBASIC applications to immediately examine what's being discussed; which is great, but what are we to do? Who has a GWBASIC interpreter anymore? We could use any one of a number of modern BASIC interpreters, but I settled on QBASIC64. It's a wonderful piece of technology that's free and looks open source.

So I'm going to do all of the suggested exercises, and see if I can expand upon them, as well as do them in both QB64 and Matlab/Octave.

The first program is "factorial.bas", which just computes the factorial in the traditional recursive way. Adorable. This wasn't in the book, but I thought for exposition it had to be added - plus more experience with BASIC/QB64. 


REM Factorial computation, in the traditional recursive style
DIM a
a = Factorial!(5)
PRINT "Factorial(5):"; a
    
    
FUNCTION Factorial! (N!)
IF N! = 0 THEN Factorial! = 1 ELSE Factorial! = N! * Factorial!(N! - 1)
END FUNCTION

There's obviously some point to reading SICP, because if you see this you should recognize that it can be transformed (either by tail-call recursion or manually) into a simple for loop: 

 
REM Factorial computation in a for loop style
DIM a
a = Factorial!(5)
PRINT "Factorial(5):"; a


FUNCTION Factorial! (N!)
    REM Declare an accumulation variable
    DIM ACC
    ACC = 1
    FOR I = 0 TO N
        IF I = 0 THEN
            ACC = ACC
        ELSE
            ACC = I * ACC
        END IF
    NEXT I
    Factorial! = ACC
END FUNCTION
